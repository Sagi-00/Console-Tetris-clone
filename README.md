# Console Tetris Clone

A Tetris clone written in C# as a learning project.

## Features

- Random Tetromino spawning
- Piece movement
- Rotation system
- Collision detection
- Line clearing
- Score system
- Game over detection

## Technologies

- C#
- .NET Console Application

## What I Learned

- 2D Arrays
- Object-Oriented Programming
- Collision Detection
- Game Loops
- Stopwatch Timing
- Pseudocode Design
- Refactoring and Single Responsibility Principle
