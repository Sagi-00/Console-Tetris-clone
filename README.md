# Console Tetris Clone

A Tetris clone written in C# as a learning project.

## Project Goal

The goal of this project was not to create a complete version of Tetris, but to practice fundamental programming concepts such as:

- Object-Oriented Programming (OOP)
- Working with 2D Arrays
- Collision Detection
- Game Loops
- Pseudocode Design
- Refactoring and the Single Responsibility Principle

The project was built as a learning exercise focused on understanding how the core mechanics of Tetris work and how to structure a larger program.

## Features

- Random Tetromino spawning
- Piece movement
- Rotation system
- Collision detection
- Line clearing
- Scoring system
- Game over detection

## Technologies

- C#
- .NET Console Application

## Rotations

In this Tetris clone, I calculated piece rotations using mathematics instead of predefined rotation states or pivot points.

## Missing Features

In this project, I wanted to focus on the core gameplay mechanics of Tetris rather than implementing every feature. Because of this, some features are missing.

I believe I learned everything I wanted to learn from this project, and that starting a new project will be more beneficial for improving both my coding skills and problem-solving abilities.
